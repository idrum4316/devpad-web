# devpad-web
This is the web component of Devpad. It's designed to be installed into and
served from the wwwroot directory of devpad/server.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
